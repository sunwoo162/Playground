# 모의 투자 관심 종목 확장프로그램

확장 아이콘을 누르면 놀이터 모의 투자에 등록한 관심 종목만 작은 팝업으로 보여줍니다.

## 개발자 모드 설치

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 우측 상단 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램 로드`를 누릅니다.
4. 이 폴더를 선택합니다: `apps/mock-invest-extension`

## 설정

기본 주소는 아래 값입니다.

```text
https://playground.https.gsmsv.site/apps/mock-invest/
```

주소가 달라지면 확장프로그램 옵션에서 서비스 주소를 수정하세요.

## 사용 조건

놀이터 웹에서 로그인된 Chrome 프로필이어야 관심 종목 API를 읽을 수 있습니다.
